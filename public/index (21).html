Submit Player Registration

Submitted to player_league_registrations with status = 'pending'. guard_registration_approval trigger prevents non-league-admin from approving directly.

Report Player Injury

Writes to player_injuries. Clinical fields (diagnosis, treatment_notes, medical_notes) visible only to physiotherapist + club_admin. Setting cleared_at triggers auto_clear_injury() which sets is_active=false.

Issue Manual Suspension

Writes to suspensions. For auto-suspensions from red cards and yellow accumulation, see auto_suspend triggers. This form is for manual_admin and serious_misconduct bans.

New Transfer Request

Club admin submits with status='pending'. guard_transfer_status_change() prevents status escalation. On approval, on_transfer_approved() sets players.club_id = to_club_id.

File Appeal

Inserts into disciplinary_appeals. chk_appeal_has_subject CHECK constraint: either suspension_id OR disciplinary_record_id must be set. suspension_held=true allows player to play pending decision.

Edit Club Details

Add Staff Member

Inserts into club_staff. If profile_id is set, that profile gains physiotherapist access to player_injuries for this club. UNIQUE(profile_id, club_id, role) when is_active=true.
